VS Code 1.124 kom 10. juni med et tydelig agentfokus. Målet er å gjøre det enklere å jobbe på tvers av mange Copilot-sesjoner uten å miste oversikt eller kontekst.

Autopilot er nå på som standard og skal selv avgjøre når en oppgave er ferdig. I tillegg kan du sende nye session-forespørsler i bakgrunnen, så du kan fortsette å skrive mens agenten jobber.

Release notes løfter også frem bedre navigasjon mellom sessions, restore on reload, browser history og enterprise-managed plugin policies. Det er små detaljer, men de gjør agentarbeidet mer praktisk i hverdagen.

For Nav er dette relevant fordi det peker mot mer moden agentbruk: flere parallelle oppgaver, bedre sporbarhet og færre avbrudd i flyten.
